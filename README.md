# NatureX

자연에서 영감을 받은 Next.js 기반 웹 프로젝트입니다.

## 기술 스택

- **Framework**: [Next.js 15](https://nextjs.org)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Font**: Geist (Vercel)

## 브랜치 구조

| 브랜치                 | 설명                |
| ---------------------- | ------------------- |
| `main`                 | 배포용 안정 버전    |
| `testing`              | 개발 및 테스트 작업 |
| `feature/landing-page` | 랜딩 페이지 작업    |

## 시작하기

```bash
# 패키지 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 으로 접속하세요.

## 배포

[Vercel](https://vercel.com)을 통해 배포합니다.
